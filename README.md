# TypeScript Modules

## Check rules that are in conflict with Prettier
```
$ npm run tslint-check
```

## TSLint and Prettier
```
$ npm run lint
```

fix command.
```
$ npm run lint -- --fix
```

## Jest unit test
```
$ npm run test
```

watch command.
```
$ npm run test -- --watch
```

## TypeDoc
```
$ npm run typedoc
```

## TSLint and Jest and TypeDoc
```
$ npm run precommit
```
